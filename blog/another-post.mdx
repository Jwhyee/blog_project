# 깃 블로그 만들기
오늘은 **Gatsby**를 사용해서 블로그를 만들어 볼 것이다.
<hr/>
<pre>
<code>
// 갯츠비 전역 설치<br/>
npm install -g gatsby-cli

// 원하는 폴더에 프로젝트 폴더 생성
cd /Desktop
mkdir gatsby_projects
cd gatsby_projects

// 새로운 개츠비 프로젝트 생성
gatsby new
</code>
</pre>
<hr/>
우선 위와 같이 작업 폴더를 생성해 Gatsby 프로젝트를 생성해준다.
<br/>
다음 VScode를 이용하여 해당 폴더를 열어준다.
